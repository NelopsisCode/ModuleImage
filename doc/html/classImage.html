)

Dessine un rectangle plein de la couleur dans l'image.

Change tout les pixels des coordonnées min aux coordonnées max par le pixel en paramètre.

Exemple de code

dessinerRectangle(1,2,5,7,couleur);

Attention aux coordonnées : Xmax (resp. Ymax) doit être plus grand que Xmin (resp. Ymin). Exemple d'erreur

dessinerRectangle(5,7,1,2,couleur);
Parameters
[in]XminLa coordonnée X du premier Pixel du rectangle
[in]YminLa coordonnée Y du premier Pixel du rectangle
[in]XmaxLa coordonnée X du dernier Pixel du rectangle
[in]YmaxLa coordonnée Y du dernier Pixel du rectangle
[in]couleurLa couleur du rectangle

void Image::effacer ( const Pixel &  couleur)

Efface l'image.

Efface l'image en la remplissant de la couleur en paramètre.
La procédure appelle la procédure dessinerRectangle() avec:
Xmin = 0 Xmax = dimx (resp. Y)

Exemple de code

[in]couleurCouleur utilisée pour recouvrir l'image.
